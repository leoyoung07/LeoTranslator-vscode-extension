# leo-translator README

This is the README for extension "leo-translator".

## Usage

- Translate And Insert

    1. Press F1.

    1. Type "translate", choose "Translate And Insert".

    1. Keybinding: `Alt + i`.

- Translate And Replace

    1. Select text.

    1. Press F1.

    1. Type "translate", choose "Translate And Replace".

    1. Keybinding: `Alt + t`.
